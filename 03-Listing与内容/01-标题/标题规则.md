# 标题规则

## 已确认规则：标题核心任务

标题需要同时完成四个任务：

1. 让消费者快速识别产品是什么。
2. 覆盖核心搜索词。
3. 明确尺寸、形状、灯光、底座等关键版本信息。
4. 避免关键词堆砌和参数误导。

标题不是卖点仓库，前半段优先保留影响点击和版本判断的信息。

## 当前标题结构拆解

当前变体标题基本遵循：

`Brand + Size + Outdoor Cantilever Patio Umbrella + 是否带底座 + LED Lights + Large/Heavy Duty + 2-Tier + Aluminum Offset Umbrella + 360° Rotation + 场景 + Color`

## 已确认规则：推荐标题信息顺序

推荐顺序：

1. 品牌。
2. 尺寸。
3. 形状。
4. 核心产品类型。
5. 灯光或核心功能。
6. 结构与材质。
7. 旋转等关键操作功能。
8. 主要使用场景。

建议结构：

`JZAC & QZYD + Size + Rectangular/Square + Cantilever Patio Umbrella + with LED Lights + 2-Tier Aluminum Offset Umbrella + 360° Rotation + 场景`

## 必须保留的核心词

- JZAC & QZYD
- 9x9 / 9x12 / 10x13
- Cantilever Patio Umbrella
- LED Lights / Solar LED Lights
- Offset Umbrella
- 2-Tier
- Aluminum
- 360-Degree Rotation
- Patio / Pool Deck / Backyard / Garden

## 已确认规则：产品类型词优先级

核心名称优先级：

1. Cantilever Patio Umbrella
2. Offset Patio Umbrella
3. Large Patio Umbrella
4. Rectangular Cantilever Umbrella
5. Square Cantilever Umbrella

不要把 `Roman Umbrella` 或 `Roman Style Umbrella` 作为 Amazon 美国站核心产品名，因为这不是美国消费者搜索此类产品的主流表达。

## 带底座规则

- 带底座变体可写 `with Base Included` 或 `Weighted Base Included`。
- 不带底座变体不能写 `Base Included`。
- 不带底座但含十字底座时，可写 cross base / steel cross base，但要避免误导为重型填充底座。
- Cross Base 不等于 Weighted Base，不能仅因为包装中有 Cross Base 就写 Base Included。
- 不带配重底座版本标题通常无需写 `No Base Included`，底座不包含的信息应在五点、图片和详情页前置说明。

## 优化建议

- 9x9 标题强调 square、solar LED、pool deck/backyard。
- 9x12 标题强调 rectangular、large shade、outdoor dining。
- 10x13 标题强调 massive rectangular shade、poolside、deck。
- 标题里不要堆砌过多重复词，重点让尺寸、底座、灯光、结构和场景一眼可见。

## 条件规则：分尺寸标题差异

### 9x9

- 应突出 `9x9 FT`、`Square`、`Cantilever Patio Umbrella`、`Solar LED Lights`、`2-Tier`、`360-Degree Rotation`。
- 带底座版本必须写 `Base Included`。
- 不带底座版本不得写 `Base Included`。

### 9x12

- 应突出 `9x12 FT`、`Rectangular`、`Cantilever Patio Umbrella`、`LED Lights`、`2-Tier`、`Aluminum`、`360° Rotation`。
- 带底座和不带底座必须明确区分。

### 10x13

- 应突出 `10x13 FT`、`Rectangular`、`Cantilever Patio Umbrella`、`LED Lights`、`Large`、`2-Tier`、`Aluminum Offset Umbrella`、`360° Rotation`。
- 可使用 `Poolside`、`Deck` 等大空间场景词。
- 10x13 属于大尺寸高利润产品，标题应强化大遮荫范围和高端大空间定位，但不能使用未经证实的绝对化表达。

## 已确认规则：关键词控制

标题中关键词应自然覆盖，不应重复堆积。

不推荐：

`Cantilever Patio Umbrella Offset Umbrella Large Patio Umbrella Outdoor Umbrella`

更推荐：

`Rectangular Cantilever Patio Umbrella with LED Lights, Large 2-Tier Aluminum Offset Umbrella`

同义词只需要自然覆盖一次。

## 已确认规则：标题禁区

不得自行加入：

- 未确认防风等级。
- 未确认认证。
- 未确认防水等级。
- 未确认伞布克重。
- 未确认 LED 数量。
- 未确认电池容量。
- 未确认续航。
- 未确认配重重量。
- `Commercial Grade` 等无依据等级。
- `100% Waterproof`。
- `All-Weather`。
- `Hurricane Proof`。
- `Windproof`。

避免过多绝对化词语：`Best`、`Perfect`、`Ultimate`、`Completely Waterproof`、`Never Fade`、`Unbreakable`。

## 已确认规则：标题优化检查清单

- 消费者能否在前半段识别尺寸和产品类型。
- 是否能立即识别带底座或不带底座。
- 核心词是否自然出现。
- 是否有明显关键词堆砌。
- 是否混入其他尺寸参数。
- 是否改变原有产品功能。
- 是否加入未经确认的信息。
- 是否适合美国消费者阅读。
- 是否保留主要转化信息。
- 标题长度增加是否真的带来信息价值。
