# 后台Search-Term

## 已确认规则：主要作用

后台 Search Terms 用于补充标题和五点中没有自然覆盖、但与产品高度相关的搜索表达。它不是把标题和五点全部复制一遍。

## 已确认规则：优先收录内容

优先放入：

- 同义词。
- 产品类型补充词。
- 使用场景词。
- 消费者常用表达。
- 长尾组合词中的重要词根。
- 标题未覆盖的相关功能词。
- 美式常用替代表达。

历史使用方向：

`sunshade parasol double canopy crank lift adjustable tilt outdoor dining table patio sofa backyard garden terrace courtyard pool seating large shade coverage`

## 已确认规则：去重

后台词不应机械重复标题中已经出现的完整词组。

例如标题已有 `10x13 FT Rectangular Cantilever Patio Umbrella with LED Lights`，后台无需再次完整写 `10x13 rectangular cantilever patio umbrella with led lights`。

更适合补充：`sunshade parasol double canopy crank lift adjustable tilt outdoor dining table patio sofa backyard garden terrace courtyard pool seating`

## 条件规则：不同版本关键词

### 带底座版本

可使用：

- patio umbrella with base included
- cantilever umbrella with weighted base
- offset umbrella with stand

### 不带底座版本

不得在后台词中放入可能引发误导的：

- base included
- with weighted base
- umbrella with base

不带底座版本可使用：

- cross base
- deck mount
- weighted base sold separately

前提是这些词符合实际安装方式。

## 已确认规则：Search Terms质量标准

关键词应满足：

- 与产品高度相关。
- 消费者搜索意图一致。
- 不造成版本误导。
- 不重复浪费空间。
- 不加入竞品品牌词。
- 不加入其他 ASIN 或品牌名称。
- 不加入主观夸张词。
- 不加入无意义词组。
