# SP关键词广告

## 适用场景

用于规范 `SP关键词广告` 相关的广告规则。

## 输入信息

| 输入 | 来源 | 是否必需 | 备注 |
|---|---|---|---|
| 关键词清单 | 搜索词报告 / 关键词库 / 竞品观察 | 是 | 用于建立 SP 关键词广告 |
| 匹配方式 | 广告后台 | 是 | 精准、词组、广泛 |
| 承接变体 | 广告结构 / 产品事实库 | 是 | 确认尺寸、底座、灯光、颜色是否匹配 |

## 已确认规则

- SP 关键词广告每周需要补充新词，尤其是精准长尾词组。
- 精准长尾词组应优先用精准匹配或词组匹配测试。
- 关键词必须由能承接的尺寸、底座、灯光和场景变体投放。

## 条件规则

| 条件 | 动作 | 例外 |
|---|---|---|
| 关键词为高相关精准长尾词 | 优先加入精准匹配 | 流量过小可同时用词组匹配测试 |
| 关键词为场景长尾词 | 加入对应场景广告组或词组匹配 | 如果场景不明确，先低预算测试 |
| 关键词包含带底座需求 | 只投放带底座变体 | 不带底座变体不得承接误导流量 |
| 关键词包含尺寸或形状 | 投放对应尺寸/形状广告组 | 不匹配尺寸不得混投 |

## 禁止事项

- 不要把精准长尾词混进无法承接的变体广告组。
- 不要为拓词而加入低相关大词。
- 不要忽略带底座和不带底座的承接边界。

## 输出结果

- 新增 SP 精准长尾词组。
- 新增 SP 词组匹配词组。
- 暂不投放或待确认的关键词。

## 复盘指标

| 指标 | 观察周期 | 判断方式 |
|---|---|---|
| Impressions | 最近 7 天 | 判断是否有展示 |
| Clicks | 最近 7 天 | 判断是否有点击 |
| Orders | 最近 7 天 | 判断是否出单 |
| ACOS / CVR | 最近 7 天 | 判断效率和承接 |

## 待验证认知

- 精准长尾词组的起始竞价和测试预算。
