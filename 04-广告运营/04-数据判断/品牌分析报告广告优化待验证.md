# 品牌分析报告广告优化待验证

## 状态

- 状态：待验证
- 记录日期：2026-06-17
- 来源：用户运营想法
- 适用模块：广告运营、Brand Analytics、品牌分析、搜索词、竞品点击份额、转化承接、Listing 优化

## 原始想法

分析亚马逊卖家后台的品牌分析里的各个报告数据，用来优化广告。

## 核心假设

卖家后台 Brand Analytics / 品牌分析中的搜索、点击、购买、竞品和品牌表现数据，可能帮助广告复盘不只停留在广告后台的 spend、clicks、orders、ACOS，而是补充判断：

- 哪些搜索词有市场需求，但广告没有覆盖或覆盖不足。
- 哪些词点击份额或购买份额低，需要广告或 Listing 承接加强。
- 哪些竞品/ASIN 在同一搜索词下抢走点击或购买，可以反推商品投放、价格或图片优化。
- 哪些词虽然广告 ACOS 不好，但从品牌分析看仍有类目机会，适合降竞价观察而不是直接否定。
- 哪些品牌词、核心词、长尾词应该分开看广告目标：出单、拓词、拿流量、控排名。

## 待确认报告清单

以下报告名称和字段需要以当前 Seller Central 后台实际可见页面为准，不能在未截图或未导出前写成固定事实。

| 报告/数据方向 | 可能用途 | 待确认字段 |
|---|---|---|
| Search Query Performance / 搜索查询表现 | 判断搜索词市场需求、品牌点击份额、品牌购买份额、转化漏斗 | 搜索量、展示、点击、加购、购买、品牌份额 |
| Search Catalog Performance / 搜索目录表现 | 判断具体 ASIN 在搜索流量中的曝光、点击、加购和购买承接 | ASIN、搜索展示、点击、加购、购买、转化率 |
| Top Search Terms / 热门搜索词 | 拓展高相关关键词和趋势词 | 搜索词排名、点击份额、购买份额、Top ASIN |
| Market Basket Analysis / 购物篮分析 | 找互补品和搭配场景，辅助商品投放和 A+ 场景 | 同购商品、同购频率、组合关系 |
| Repeat Purchase Behavior / 重复购买行为 | 判断复购弱/强，但罗马伞大件低频复购，广告使用价值需谨慎 | 复购率、复购周期、购买人数 |
| Brand Metrics / 品牌指标 | 判断品牌认知、考虑、购买漏斗是否改善 | 品牌指标、类目对比、漏斗阶段 |

## 可验证广告动作

### 1. 用搜索查询表现找广告覆盖缺口

| 判断 | 广告动作 | 验证方式 |
|---|---|---|
| 品牌购买份额低，但搜索词高度相关 | 加入精准/词组候选，或提高相关广告组预算 | 对比加词后 `7-14` 天 clicks、orders、ACOS、CVR |
| 品牌点击份额低，但购买份额相对好 | 可能需要提高曝光或点击入口 | 测试竞价、广告位、主图/标题承接 |
| 点击份额高但购买份额低 | 可能是价格、Listing、变体或页面承接问题 | 不急于加预算，先查 CVR、价格、Coupon、Review、变体 |

### 2. 用搜索目录表现检查 ASIN 承接

| 判断 | 广告动作 | 验证方式 |
|---|---|---|
| 某 ASIN 搜索点击多但购买弱 | 减少该 ASIN 在相关词的投放或降竞价 | 对比调整后 ACOS、CVR、订单变化 |
| 某 ASIN 购买承接好但广告投放弱 | 增加该 ASIN 对应词和广告组覆盖 | 对比 `7-14` 天订单和自然排名变化 |
| 同一词下不同变体承接差异大 | 调整广告变体组合 | 对比不同变体广告订单、CVR、ACOS |

### 3. 用热门搜索词和竞品份额拓词

| 判断 | 广告动作 | 验证方式 |
|---|---|---|
| 高相关长尾词有购买份额但当前广告未覆盖 | 加入手动精准/词组 | 观察 `7-14` 天点击、订单、ACOS |
| Top ASIN 与我方产品高度相似 | 加入商品投放或竞品页面测试 | 观察商品投放点击、订单、ACOS |
| 大词流量高但相关性偏泛 | 不直接重预算抢大词，先用低预算测试 | 防止浪费预算 |

### 4. 用购物篮分析找商品投放和场景机会

| 判断 | 广告动作 | 验证方式 |
|---|---|---|
| 与户外家具、庭院桌椅、伞座等高相关商品同购 | 测试相关 ASIN 商品投放或场景词 | 观察商品投放订单、ACOS |
| 同购商品与罗马伞需求弱相关 | 不作为广告拓展方向 | 避免低相关流量 |

## 与现有广告规则的关系

- 与 [[09-规则案例与实验/06-已验证结论/广告-复盘步骤与数据周期选择]] 兼容：品牌分析数据用于辅助判断，但广告动作仍需按 `3` 天、`7` 天、`14-30` 天周期验证。
- 与 [[09-规则案例与实验/01-待验证假设/README]] 中“先按广告目标分析”兼容：不同报告应先服务不同目标，例如拓词、控排名、出单、拿流量。
- 与 [[04-广告运营/04-数据判断/AMC营销云搜索词分析]] 互补：AMC 更偏广告触达和转化路径，品牌分析更偏搜索需求、点击份额、购买份额和竞品对比。

## 风险边界

- 品牌分析不能直接替代广告后台数据，必须和 spend、clicks、orders、sales、ACOS、CPC、CTR、CVR 交叉验证。
- 品牌分析中的高流量词不等于一定适合投放；仍需判断与罗马伞尺寸、底座、灯光、场景是否匹配。
- 竞品点击份额或购买份额高，不等于我方应该无条件抢同一词；需要看价格、Review、图片、配送、Coupon 和变体承接。
- 报告名称、字段和权限可能随 Seller Central 后台变化；未导出前不得写成固定字段事实。
