# 否定关键词规则

## 适用场景

用于规范 `否定关键词规则` 相关的广告规则。

## 输入信息

| 输入 | 来源 | 是否必需 | 备注 |
|---|---|---|---|
| 搜索词报告 | Amazon 广告后台 / 领星 | 是 | 用于判断真实搜索词 |
| 搜索词相关性 | 人工判断 / 知识库关键词规则 | 是 | 判断是否与产品、尺寸、底座、灯光、场景匹配 |
| 点击次数 | 广告报表 | 是 | 判断是否属于高点击无单 |
| 广告订单数 | 广告报表 | 是 | 判断是否出单 |

## 已确认规则

- 每周必须否定相关度低的搜索词。
- 每周必须检查有大量点击但不出单的关键词。
- 低相关词可以优先否定；高相关但无单词不能机械否定，需要先判断价格、Listing、变体、广告位和竞价是否影响转化。
- 投放词尽量不轻易否定；如果表现不好，优先考虑降低竞价。
- 能出单的词很多，约 `30` 天数据表现仍较差，或 `7-14` 天点击次数多但订单数据差的词，可以否掉。
- 不用过度担心否掉主流词；如果后续广告组数据变差，可以再考虑把这个词重新放出来。

## 条件规则

| 条件 | 动作 | 例外 |
|---|---|---|
| 搜索词与罗马伞、尺寸、底座、灯光、使用场景明显不相关 | 加入否定关键词 | 如果该词历史有明确订单贡献，先复查匹配关系 |
| 搜索词大量点击但无单，且相关性低 | 加入否定关键词 | 无 |
| 搜索词大量点击但无单，但相关性高 | 暂不直接否定，先复查价格、Listing、变体、广告位和竞价，并优先考虑降竞价 | 如果 `7-14` 天点击多但订单差，或约 `30` 天表现仍差，再考虑否定 |
| 搜索词可能造成版本误导，如底座、尺寸、灯光不匹配 | 优先否定或拆分到正确变体广告 | 如果页面已清楚承接该需求，可保留观察 |

## 禁止事项

- 不要只因 7 天无单就机械否定高相关核心词。
- 不要将带底座需求词投给无法承接的非带底座变体后继续放任消耗。
- 不要否定尚未达到最低判断数据量的高相关词，除非明显不相关。

## 输出结果

- 每周否定关键词清单。
- 高点击无单但暂不否定的观察清单。
- 高相关但优先降竞价而非否定的词清单。
- 可考虑重新放出的历史否定词清单。
- 因版本误导需要拆分或调整承接变体的搜索词清单。

## 复盘指标

| 指标 | 观察周期 | 判断方式 |
|---|---|---|
| 搜索词相关性 | 每周 | 与产品、尺寸、底座、灯光、场景是否匹配 |
| 点击次数 | 最近 7 天 | 判断是否有足够点击 |
| 广告订单数 | 最近 7 天 | 判断是否出单 |
| ACOS / Spend | 最近 7 天 | 判断消耗压力 |
| 高相关词否定判断 | 7-14 天 / 30 天 | 判断是否已经达到否词周期 |
