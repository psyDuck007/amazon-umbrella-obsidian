# 高点击无转化处理

## 适用场景

用于规范 `高点击无转化处理` 相关的广告规则。

## 输入信息

| 输入 | 来源 | 是否必需 | 备注 |
|---|---|---|---|
| 搜索词 / 投放词 | 广告报表 | 是 | 判断高点击无转化对象 |
| 点击次数 | 广告报表 | 是 | 判断是否达到处理阈值 |
| 订单数 | 广告报表 | 是 | 判断是否无转化 |
| 价格 / Coupon | Amazon 前台 / 后台 | 是 | 判断是否价格承接弱 |
| 变体承接 | 广告结构 / Listing | 是 | 判断是否投错颜色、尺寸或底座版本 |

## 已确认规则

- 高点击无转化每周必须处理，但处理动作不等于一律否定。
- 低相关高点击无转化词优先否定。
- 高相关高点击无转化词需要先诊断价格、Listing、变体承接和广告位。

## 条件规则

| 条件 | 动作 | 例外 |
|---|---|---|
| 低相关且高点击无转化 | 否定关键词 | 无 |
| 高相关但高点击无转化 | 检查价格、Coupon、Listing、变体和广告位 | 如果连续多周期无改善，可降竞价或否定 |
| 有点击但投放变体不匹配 | 更换或增加广告变体 | 如果当前变体是唯一可售承接，先调 Listing 或价格 |
| 有点击但竞品价格更强 | 考虑降价、Coupon 或促销 | 如果利润不足，先控竞价或减少预算 |

## 禁止事项

- 不要只看无订单就直接否定所有高相关词。
- 不要忽略广告归因延迟和最近 7 天样本量。
- 不要把价格、变体承接问题误判成关键词问题。

## 输出结果

- 需要否定的低相关高点击无单词。
- 需要继续观察或优化承接的高相关高点击无单词。
- 需要调整价格、Listing、变体或广告位的对象。

## 复盘指标

| 指标 | 观察周期 | 判断方式 |
|---|---|---|
| Clicks | 最近 7 天 | 判断是否高点击 |
| Orders | 最近 7 天 | 判断是否无转化 |
| CVR | 最近 7 天 | 判断承接能力 |
| ACOS / Spend | 最近 7 天 | 判断消耗压力 |

## 待验证认知

- 不同尺寸和广告类型的高点击阈值。
