# 第二箱仍在运输

## 适用场景

买家只收到一箱，另一箱 tracking 显示仍在运输、延迟、预计稍后派送。

## 已确认事实

- 罗马伞与底座可分箱发货。
- 分箱包裹可能不同日期送达。
- 不能把物流延迟直接说成丢件，除非 tracking 或承运商证据支持。

## 处理步骤

1. 核对两个 tracking number 是否都属于同一订单。
2. 判断已送达包裹是伞体还是底座。
3. 查看未送达包裹状态：运输中、派送中、异常、无更新。
4. 若仍在正常运输，向买家解释分箱并提供预计时间。
5. 若超过合理等待时间或出现异常，转入补发/索赔判断。

## 客服回复要点

- 不让买家重复解释问题。
- 给出明确的未到包裹状态。
- 如需继续等待，要写清楚后续跟进节点。

## 英文模板

We checked the shipment and found that the order was sent in separate packages. One package has been delivered, and the other package is still in transit.

This can happen with oversized patio umbrella orders because the umbrella and base are handled as separate packages. Please allow the carrier a little more time to complete the delivery. We will continue monitoring the tracking and help you if there is any delivery exception.

## 待确认

- 未送达包裹 tracking。
- 是否有承运商异常记录。
- 是否需要设定再次跟进时间。
