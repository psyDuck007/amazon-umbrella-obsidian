# 沟通合规规则

## 总原则

所有售后邮件必须在解决问题的同时避免触发亚马逊违规风险。客服表达要保持书面、客观、可证明，不绕过平台、不诱导评价、不承诺未经确认或平台不允许的补偿。

## 绝对禁止

- 严禁诱导买家留下、修改或删除 Review。
- 严禁要求买家先删除差评再提供退款、补发或其他补偿。
- 严禁为了普通售后故障排除私下拨打买家电话。
- 严禁引导买家跳出亚马逊站内信、订单系统或平台允许渠道沟通。
- 严禁夸大防风、防水、认证、质保、承重、灯光、电池等未确认信息。
- 不得使用威胁、催促、暗示惩罚或让买家承担不合理义务的表达。
- 涉及退款、退货、补发、索赔时，必须保留书面记录和可追溯依据。

## 可以表达

- 可以礼貌致歉并说明会继续协助处理。
- 可以请求买家提供必要照片，用于确认包装、破损、缺件或物流索赔。
- 可以解释大件商品分箱派送，且不同包裹可能在不同日期送达。
- 可以说明退款通常需要 `3-7 个工作日` 回到原付款账户。
- 可以说明为了遵守亚马逊沟通准则，需要继续通过亚马逊消息系统处理。
- 可以提醒庭院伞在起风天气应收拢，且带底座产品需要正确加重；不得写防风等级。

## 高风险表达替换

| 不要写 | 改成 |
|---|---|
| Please change/remove your review and we will refund you. | We will assist with the issue through this order message. |
| Call us at... / text us... | To keep a complete record, we will continue assisting through Amazon messages. |
| This umbrella is windproof. | Please close the umbrella during windy conditions and make sure the base is properly weighted. |
| We guarantee it will never tip over. | Proper setup and sufficient base weight are important for stable use. |
| We can offer compensation if you keep the item and leave positive feedback. | We can offer a partial refund as a solution for this issue. |

## 电话联系规则

如果买家留下电话并要求回电，默认不能拨打私人电话。回复时应说明：为了遵守亚马逊沟通准则并保留完整处理记录，我们会继续通过亚马逊消息系统协助解决问题。

## 发送前合规检查

- 是否出现 Review、Feedback、rating、stars 与退款/补发绑定。
- 是否出现站外链接、私人邮箱、私人电话、非平台支付路径。
- 是否承诺未确认质保、认证、防风、防水、电池、承重或配件。
- 是否要求买家承担不合理义务。
- 是否保留了下一步处理动作和证据依据。
