# 退货标签规则

## 适用场景

买家坚持退货、商品无法通过补发/退款挽留解决，或平台流程需要买家退回商品。

## 已确认规则

- 如买家坚持退货，应顺畅提供 FedEx 预付费退货标签。
- 分箱商品通常需要分别匹配每个包裹的标签。
- 如果退回底座，必须提醒底座不能加水或沙；已加水或沙需先排空。
- 发放标签前默认要求 2-3 张照片，用于确认包装状态、匹配退货仓库和物流索赔。
- 买家抱怨大件难搬运时，可提示可使用 FedEx 预付费标签预约上门取件。

## 处理步骤

1. 确认退货对象：整套、伞体箱、底座箱或配件。
2. 确认是否分箱，避免只给一张标签。
3. 索要照片确认包装状态和包裹标签。
4. 如退底座，提醒不得带水/沙退回。
5. 发放对应 FedEx 预付费标签。
6. 告知买家可在 FedEx 官网预约上门取件。

## 英文模板

We can help arrange a prepaid FedEx return label for you. Because this is an oversized item, please make sure the item is packed securely before return.

If the base is included in the return, please do not fill it with water or sand. If it has already been filled, please empty it completely before packing it for return.

You may also use the prepaid FedEx label to schedule a pickup through FedEx if the package is difficult to carry.

## 待确认

- 该订单需要几张退货标签。
- 退货仓库和承运商规则。
- 底座是否已经注水或加沙。
