# 全额与部分退款

## 适用场景

买家提出退款、商品存在轻微问题、退货成本过高、或需要用退款方案降低大件退货损失。

## 已确认规则

- 部分退款可用于非核心功能故障、轻微外观问题、包装凌乱但商品可用、买家愿意保留商品等场景。
- 参考区间通常为 `$30-$50`，需根据问题严重程度、订单金额和利润空间调整。
- 全额退款适用于严重破损、核心功能无法使用、无法补发解决、或平台/业务判断必须全额处理的场景。
- 退款不得与 Review、Feedback、评分、删评或改评绑定。
- 发放退款后需提醒 `3-7 个工作日` 退回原付款账户。
- 退款未成功、退款状态不清楚或买家多次追问时，按高风险订单处理：核对退款金额、发起日期、后台状态和原支付路径，并给出下一次检查时点。
- 买家提出较大金额退款时，应加强证据和授权核对，但不能仅凭金额认定买家恶意。

## 判断规则

| 场景 | 优先方案 | 备注 |
|---|---|---|
| 轻微外观瑕疵，不影响使用 | 部分退款 | 先询问买家是否愿意保留 |
| 包装破损但商品可用 | 部分退款 | 保留照片用于物流索赔 |
| 配件缺失但可补发 | 补发配件 | 退款不是首选 |
| 核心结构损坏 | 补发或全额退款 | 根据照片判断 |
| 买家坚决退货 | 免费退货标签 | 分箱商品需匹配标签数量 |

## 英文模板：部分退款

We are sorry for the inconvenience. If you are willing to keep the item, we can offer a partial refund as a solution for this issue.

Once the refund is issued, it usually takes 3-7 business days to return to the original payment method.

## 英文模板：全额退款

We are sorry that this issue could not be resolved as expected. We can help arrange a full refund according to the order handling process.

Once the refund is issued, it usually takes 3-7 business days to return to the original payment method.

## 待确认

- 当前订单利润和可接受退款上限。
- 是否已有照片或其他证据支持退款金额。
- 是否需要退货后退款，还是可免退货退款。
