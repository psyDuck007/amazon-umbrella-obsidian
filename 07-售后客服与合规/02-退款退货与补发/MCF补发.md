# MCF补发

## 适用场景

通过 Amazon MCF 或其他仓库渠道为买家补发配件、单箱或整套。

## 使用前提

- 必须确认原订单 ASIN、尺寸、颜色、是否带底座。
- 必须确认补发内容与买家实际问题匹配。
- 必须确认库存和发货仓库，避免发错版本或重复发货。
- 补发不得与 Review、Feedback 或评分绑定。

## 处理步骤

1. 明确补发类型：配件、伞体箱、底座箱、整套。
2. 核对对应 SKU / ASIN / 仓库库存。
3. 创建 MCF 或补发单。
4. 保存补发单号和 tracking。
5. 给买家发送补发确认，不承诺未确认送达日期。
