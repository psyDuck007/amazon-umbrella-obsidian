# 价格-非主卖颜色加价与低库存展示

## 状态

- 结论状态：已验证
- 记录日期：2026-06-11
- 来源：用户运营经验
- 适用模块：价格促销与利润、库存与物流、颜色变体运营

## 原始结论

针对颜色变体，如果某个颜色不是主卖变体，且当前销量表现不好，可以尝试在原售价基础上加价约 `$10`，同时将后台可售库存展示控制在约 `10` 把，使前台出现库存不足或仅剩约 `10` 把的稀缺提示。用户已验证：这个动作后，该颜色变体销量反而会变好。

## 适用条件

| 条件 | 要求 |
|---|---|
| 变体类型 | 同一父体下的颜色变体 |
| 变体角色 | 非主卖颜色，不承担主流量和主转化 |
| 当前状态 | 销量较弱、点击或浏览存在但下单动力不足 |
| 价格空间 | 加价约 `$10` 后仍不明显脱离同尺寸、同配置、同颜色竞品价格带 |
| 库存边界 | 前台展示的可售数量应与真实可售计划、后台可售设置和平台规则保持一致 |

## 操作规则

1. 先确认该颜色不是当前主卖颜色，不要对核心走量颜色直接套用。
2. 检查该颜色近期销量、转化率、广告花费和库存周转，避免把缺流量问题误判为价格问题。
3. 在原售价基础上加价约 `$10`。
4. 将前台可见可售库存控制在约 `10` 把，使买家感知到库存有限。
5. 观察调整后销量、转化率、Session、广告 ACOS、自然排名和退款退货变化。
6. 如果加价后转化继续下降，或广告 ACOS 明显恶化，应回退价格或恢复库存展示。

## 可复用判断

- 非主卖颜色低价不一定能拉动转化，有时低价会强化“滞销颜色”感知。
- 适度加价可能让该颜色看起来更有价值，同时低库存展示会提高购买紧迫感。
- 该策略更适合已经有一定流量但下单犹豫的颜色，不适合完全没有曝光的颜色。

## 风险与边界

- 不得把虚假库存写进 Listing、图片、五点、客服回复或站外宣传。
- 不得为了制造稀缺感而让前台展示与真实可售计划严重不一致。
- 如果该颜色实际库存压力很大，需要同时评估清库存目标，不能只看短期转化。
- 如果竞品同色价格明显更低，加价可能损害点击率和转化率。

## 复盘指标

| 指标 | 观察周期 | 判断方式 |
|---|---|---|
| Sessions | 调整前后 7-14 天 | 判断流量是否变化 |
| Unit Session Percentage | 调整前后 7-14 天 | 判断转化是否改善 |
| 订单量 | 调整前后 7-14 天 | 判断销量是否真实提升 |
| 广告 ACOS | 调整前后 7-14 天 | 判断广告承接是否恶化 |
| 自然排名 | 调整前后 7-14 天 | 判断销量提升是否带动排名 |
| 退款退货率 | 调整后 14-30 天 | 判断加价是否引发预期落差 |
