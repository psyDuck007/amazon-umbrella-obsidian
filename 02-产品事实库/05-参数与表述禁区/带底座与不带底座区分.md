# 带底座与不带底座区分

## 当前变体区分

| 尺寸 | 不带底座 ASIN | 带底座 ASIN |
|---|---|---|
| 9x9 | B0G797B211 | B0G79MQ7YF |
| 9x12 | B0GS1TNH5G | B0GS1ZF41M |
| 10x13 | B0GS21B48Y | B0GS1XCSSF |

## 规则

- `Base Included` 只能用于带底座 ASIN。
- 不带底座 ASIN 可以写 cross base included，但不能让买家误以为含重型填充底座。
- 图片展示另售底座时，必须加 sold separately。
- 售后遇到“只收到底座/只收到伞体”时，先确认购买的是带底座还是不带底座变体。
- Cross Base 不等于 Weighted Base。不能仅因为包装中包含 Cross Base 就写 `Base Included`。
- 不带配重底座版本标题通常无需写 `No Base Included`，但五点、图片和详情页必须清楚说明 Weighted Base Not Included 或 Weighted Base Sold Separately。
- 带底座版本需要在标题、五点、图片和售后说明中保持一致，避免买家误以为漏发或多箱未送达。

## 高风险场景

- 不带底座标题误写 Base Included。
- 不带底座主图展示重型底座但没有 sold separately。
- 带底座分箱发货导致买家误以为漏发。
- 底座退货时买家已经加水/沙。

## 条件规则：底座重量

- 10x13 历史确认参数：注水约 290 lbs，湿沙约 550 lbs。该参数只能用于已确认的 10x13 大底座版本。
- 9x9 灯珠伞带底座版本和 9x12 半灯伞带底座版本：用户确认使用小底座；可按 `90%` 沙 + `10%` 水填充，正确填充后总配重约 `220 lbs`。
- 9x9 和 9x12 历史方向：不带底座版本另购 fillable weighted base 时，建议承重能力超过 220 lbs。该建议主要适用于历史确认的 9x9 和 9x12 不带底座体系。
- 重量表达应说明是填充后的总配重能力，不是空底座净重。
