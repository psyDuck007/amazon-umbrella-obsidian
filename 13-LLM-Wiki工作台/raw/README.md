# raw 原始资料层

## 用途

这里保存尚未编译成规则的原始资料，包括用户零散想法、报表摘录、竞品观察、客服案例、截图文字说明和外部资料摘要。

## 子目录

| 子目录 | 内容 |
|---|---|
| `广告` | 广告报表、搜索词、操作记录、投放观察 |
| `Listing` | 标题、五点、图片、Search Terms 原始想法 |
| `定价` | 价格、Coupon、促销、利润、价格实验原始记录 |
| `库存` | 库存展示、备货、断货、清库存、仓库相关观察 |
| `竞品` | 竞品页面、价格、广告位、评论、VOC 观察 |
| `售后` | 客服案例、退款补发、评价申诉、合规风险材料 |

## 规则

- 原始资料可以保留口语和不完整表达。
- 原始资料不等于已确认事实。
- 编译后需要更新 `13-LLM-Wiki工作台/log.md`。
- 涉及未确认产品信息时，同步进入 `12-收集箱与附件/03-待确认产品信息.md`。
