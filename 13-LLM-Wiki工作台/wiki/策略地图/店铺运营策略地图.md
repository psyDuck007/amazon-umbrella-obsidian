# 店铺运营策略地图

## 定位

本页用于把 JZAC / QZYD 罗马伞运营中的广告、Listing、定价、库存、售后和竞品信息连接起来，避免各模块各自优化但目标冲突。

## 总体链路

```text
竞品与VOC
  -> Listing定位与卖点
  -> 广告投放与搜索词
  -> 价格与促销
  -> 库存与发货体验
  -> 售后反馈与评价
  -> 反哺产品事实、Listing和广告策略
```

## 模块联动

| 模块 | 输入 | 输出 | 反哺对象 |
|---|---|---|---|
| 竞品与 VOC | 竞品价格、评论、广告位、买家痛点 | 机会点、风险点、卖点优先级 | Listing、广告、定价 |
| Listing | 产品事实、关键词、买家疑虑 | 标题、五点、图片、Search Terms | 广告、售后 |
| 广告 | 搜索词、点击、转化、ACOS、广告位 | 关键词分层、预算和竞价动作 | Listing、定价、竞品 |
| 定价 | 利润、竞品价格、转化率、库存压力 | 售价、Coupon、促销、价格实验 | 广告、库存 |
| 库存 | 可售库存、补货周期、仓库、发货方式 | 库存展示、断货预警、清库存动作 | 定价、广告 |
| 售后 | 破损、缺件、误解、评价、申诉 | 客服 SOP、Listing 修正点、产品改进点 | Listing、产品事实、库存 |
